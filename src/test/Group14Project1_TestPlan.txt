
DESCRIPTION:
-This cases are for practicing the VCS collaboration among our team.
-Clone the repository to your local repo.
-Choose a TC and make the team members know which case belongs to you. (Use comment feature)
-Create your class name respected to your test case.(TC1, TC2...)
-Everybody will create a branch and work on it. I will have the master branch. Then we will merge one by one all
 branches to the master and finally create our test case folder.
-Please practice fetch, commit, push and pull request.

PRE CONDITIONS:
- All dependencies added by me to the pom.xml file.
- .gitignore file is set and ready.
- Make sure your computer language is English.

TEST CASES:
TC1: etsy.com Verification – Section 1
1. Open Chrome browser
2. Go to https://www.etsy.com/
3. Verify the web page title is “Etsy - Shop for handmade, vintage, custom, and unique gifts for everyone"
4. Click to Etsy logo top left of the site.
5. Verify page title is not changed

TC2: etsy.com Verification – Section 2
1. Open Chrome browser
2. Go to https://www.etsy.com/
3. Print "bohemian sandals" in the search box.
4. Hit enter.
5. Check if there is a button "under 200$"
6. Click it and return the total results of the bohemian sandals.

TC3: etsy.com Verification – Section 3
1. Open Chrome browser
2. Go to https://www.etsy.com/
3. Click Jewelry & Accessories module
4. Print total result.
5. Go back.
6. Print total result in all categories. ( All categories (65,443,161 results, with Ads Learn more))
7. Verify that all categories result is bigger than Jewelry & Accesories page.
